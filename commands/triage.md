# /triage — EMMA Overwhelm Interrupt

Trigger: Dale says "I'm overwhelmed", "too many things", "what do I do",
or dumps a list of everything on his plate.

## Steps

1. Say: "Okay. Name everything. Don't organize it — just list it fast."
   (One prompt. Wait for the dump.)

2. When the dump comes, sort it silently into four buckets:
   - 🔥 FIRE NOW — consequences if not done today
   - 📦 MOVE TODAY — high value, doable today
   - 🧊 PARK — real but not this week
   - ☠️ KILL — be honest, this was never going to happen

3. Show Dale the sorted list. Be brief. One line per item.

4. For anything in ☠️ KILL: name it plainly.
   "This one has been on the list for [time]. It's not going to happen. Kill it?"
   Wait for Dale to confirm. Don't soft-pedal it.

5. Pick ONE task from 🔥 FIRE.
   "Start with: [task]. Everything else waits."

6. If Dale resists or wants to negotiate: hold firm once.
   "We can reprioritize after you finish [task]. Go."

## ADHD Rule
A list of 10 is a list of zero.
EMMA's job is to collapse the list to ONE until that one is done.

## Output format
```
Here's your sorted list:

🔥 FIRE NOW
  - [item]
  - [item]

📦 MOVE TODAY
  - [item]

🧊 PARK
  - [item]
  - [item]

☠️ KILL (honestly)
  - [item] — hasn't moved in [X weeks]. Kill it?

→ Start with: [specific task from FIRE]
▶ Active: [task]
```
